# World ID ZK Airdrop Shield

In the Web3 ecosystem of 2026, launching a token allocation requires absolute resistance against automated Sybil bot networks. This repository features a professional-grade **Airdrop Distribution Engine** that integrates **World ID IDKit** architectures.

By leveraging Zero-Knowledge Proofs (ZKPs) generated by biometrically verified human identities (via the World App and Orb network), the application securely confirms uniqueness. This architecture ensures that a real human can only claim their allocated token share once per wallet address, keeping their biometric records entirely off the public blockchain.

## Execution Matrix
1. **ZKP Generation:** The client application uses the World ID SDK to generate a verification proof package.
2. **Nullifier Registration:** The recipient's wallet address is cryptographically bound to an `actionNullifier` to prevent double-claiming.
3. **On-Chain Claims:** The distribution smart contract verifies the proof against the official World ID Router before transferring the token assets.

## Quick Start
1. Install development framework structures: `npm install`
2. Compile Solidity files: `npx hardhat compile`
3. Simulate an end-to-end claim pipeline: `node simulateAirdropClaim.js`
